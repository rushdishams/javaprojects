					PDF to Text Converter v1.0
							November 2010
                                                                       
				JAR file for converting PDF files to TXT
                                                                       
                                                                      
=====================================================================

The PDF to Text Converter converts most of the pdf files to ASCII text files.

Simply put:

java -jar PDFtoTEXT.jar <source file directory path and file name> <destination directory path and file name>

DISCLAIMER: For many reasons, a PDF file is restricted for an ASCII conversion. In that case, 
this tool may fail to perform the desired task.
